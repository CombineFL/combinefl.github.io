Evaluating and Combining Fault Localization Techniques from Different Families

An infrastructure for evaluating and combining fault localization techniques.

  • Faults dataset: Defects4j
  • Included techniques:
    • Spectrum-based fault localization
    • Mutation-based fault localization
    • (Dynamic) slicing
    • Stack trace analysis
    • Predicate switching
    • History-based fault localization

Documentation

1. Download the tool here.

2. Unzip the TAR file: 

$ tar czvf CombineFL-v0.1.tar.gz
$ cd combinefl

3. Combine techniques and Generate SVMRank file (add-in file is optional): 

$ python 1-combine.py [-c <add-in-file>]

4. Split file for k-fold validation: 

$ ./2-split.sh

5. Train and Predict in each fold: 

$ ./3-crossvalidation.sh

6. Calculate the metrics: 

$ python 4-calc-metric.py

More detailed information is available in README file.
